# ☕ Coffee Sales Analysis Dashboard (Excel)

## 📌 Project Overview
This project analyzes coffee shop sales data (149,116 transactions) and presents key business insights through an interactive Excel dashboard.

---

## 📊 Dataset Description
The dataset contains the following columns:

- transaction_id
- transaction_date
- transaction_time
- transaction_qty
- store_id
- store_location
- product_id
- unit_price
- product_category
- product_type
- product_detail
- Size
- Month Name
- Day Name
- Hour
- Sales

---

## 🛠 Data Processing Steps
- Cleaned inconsistent date and time formats
- Extracted Month, Day, and Hour from transaction data
- Created calculated column:
  
  Sales = transaction_qty × unit_price

- Verified data consistency and removed errors

---

## 📊 Dashboard Features
- Total Revenue, Transactions, and Quantity KPIs
- Hourly Sales Trend Analysis
- Top 10 Products by Quantity
- Location-wise Sales Distribution
- Product Category Distribution
- Size-wise Sales Analysis

---

## 📈 Key Insights
- Peak sales occur during morning hours (7 AM – 10 AM)
- Coffee category contributes the highest revenue
- Sales differ significantly across store locations
- Large and Regular sizes are most preferred
